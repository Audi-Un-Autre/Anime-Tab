# Anime-Tab
Anime Tab is a program that allows users to keep track of all things anime, manga, manhwa, comic, and more. Consider it the sticky notes of anime, but much more sophisticated.

<br />

# Stacks
- [x] Java 8
- [x] JavaFX 15
- [x] JavaFX FXML
- [x] MySQL
